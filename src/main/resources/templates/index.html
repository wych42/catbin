NAME
    Catbin: CLI Pastebin for /bin/cat by Spring

SYNOPSIS
    <command> | curl -F 'cat=<-' 

DESCRIPTION
    add ?<lang> or /<lang> to resulting url for line numbers and syntax
    highlighting

EXAMPLES
    ~ % curl -T bin/code.js  
       

    ~ % cat bin/code.js | curl -F 'cat=<-'  
       

    ~ % curl  
    ~ % firefox
