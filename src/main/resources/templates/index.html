NAME
    Vinergy: CLI Pastebin within VimEnergy

SYNOPSIS
    <command> | curl -F 'cat=<-' 

DESCRIPTION
    add ?<lang> or /<lang> to resulting url for line numbers and syntax
    highlighting

EXAMPLES
    ~ % curl -T bin/code.js  
       {{ url }}cbc

    ~ % cat bin/code.js | curl -F 'cat=<-'  
       {{ url }}cbc

    ~ % curl  
    ~ % wget -qO-  
    ~ % firefox
